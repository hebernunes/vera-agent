## About

Vera is a security-first AI assistant deployed via OpenClaw in Docker, 
integrated with Slack, Gmail, and Brave Search. This repo contains 
reference documentation for the setup — not operational files.

Operational files (SOUL.md, MEMORY.md, TOOLS.md etc.) live in the 
private vera-workspace repo.
